# 我的人生库

个人档案管理应用 —— 记录工作与生活的点滴。

## 功能

- **工作板块**：公众号运营 / 百度推广营销 / 视频制作 / 行政活动协助 / 工作资料
- **生活板块**：运动（健身打卡 + 爬山记录 + 饮食统计）/ 挣钱 / 爱情 / 人际
- **5色主题系统**：森语原色 / 春莓序曲 / 紫薯奶方 / 鲸落于海 / 海岛暖风
- **数据持久化**：所有数据存储在浏览器 localStorage，无需后端

## 技术

纯前端单页应用：HTML + CSS + JavaScript，零依赖。
